Using Weave with Zeta

If you want to run WeaveZZT on a modern machine, you will need an emulation environment, as Weave is a native DOS executable.

To use Zeta in Windows (the most common setup) you will need the following files from the latest zeta download at https://zeta.asie.pl/:

  • zeta86.exe
  • SDL2.dll
You will copy those into a directory, say C:\Weave, and extract the contents of your WeaveZZT download into that same directory.

Make a copy of the WEAVE3.EXE file and rename it to ZZT.EXE. The ZZT.EXE and zeta86.exe should be in the same directory.

Run zeta86.exe. If everything's working correctly, the WeaveZZT Demo world, WEAVEDMO, will come up, which shows off many enhanced features stock ZZT can't use.

Here's a quick rundown of the files you should expect to have when you play your game:

  • zeta86.exe - the emulator
  • SDL2.dll - the graphics library
  • WEAVE3.EXE - the game engine
  • WEAVEZZT.CFG - configuration file for WEAVE3.EXE, including what .ZZT file to load
  • <Game Name>.ZZT - the game file
  • <Game Name>.CFG - config options for <Game Name>.ZZT
  • <Game Name>.INI - program settings for <Game Name>.ZZT
  • <Game Name>.HLP (optional) - help file for <Game Name>.ZZT
  • <Game Name>.CHR (optional) - character set for <Game Name>.ZZT
  • <Game Name>.PAL (optional) - default palette for <Game Name>.ZZT
  • Possibly other files, including .OBJ files, additional .CHR files, and text files
You can look at WEAVEDMO for some examples of how to use .OBJ, .CFG, and .INI files in your game.
